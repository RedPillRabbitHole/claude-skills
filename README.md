# claude-skills

A collection of Claude skills for specialised assistant behaviour.

## Skills

### microsoft-learn-priority
Prioritises official Microsoft Learn documentation and provides complete, beginner-friendly step-by-step instructions for all Microsoft products. Triggered by starting your message with **"Microsoft learn"**.

Covers: Microsoft 365, Azure, Windows, Exchange, PowerShell, Entra ID, Teams, SharePoint, Intune, Purview, and all other Microsoft products and services.

### sonicwall-skill
Provides accurate, step-by-step SonicWall configuration and troubleshooting instructions based on official SonicWall documentation. Always searches official docs first and prioritises GUI over CLI.

Covers: SonicWall firewalls (TZ, NSa, NSsp, NSv series), Cloud Secure Edge (CSE) ZTNA/SASE platform, and Network Security Manager (NSM) centralised management.

Triggered by keywords: **SonicWall**, **SonicOS**, **Cloud Secure Edge**, **CSE connector**, **NSM**, **Network Security Manager**, **MySonicWall**.
